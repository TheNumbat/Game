Detailed Description

Describes an area of the map.

The mapMgr class manages all map and entity system related tasks, including storing the actual map. Each vector describes an area of the map, like a chunk. The "chunks" hold the "origional" shared_ptrs for the entities, and everything referencing entities should return a weak_ptr to these.

Each "chunk" has a hashed chunk_position value, creating a sparsely stored map.

Note
More data will probably be added to the map in the future
